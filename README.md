# m-lucky10-guide

本仓库用于归档和发布多个独立的 HTML 页面。这些页面由不同来源整理或生成，以文件形式存放在仓库中，便于访问和回溯。

## 项目简介

m-lucky10-guide 是一个静态 HTML 页面集合仓库。所有页面均以独立的 HTML 文件形式存在，不依赖特定的后端服务或数据库。仓库本身不针对任何单一域名或网站，而是作为通用存档，存放各类 HTML 内容。

## 目录说明

```
m-lucky10-guide/
├── index.html          # 仓库主页（可选）
├── pages/              # 存放主要 HTML 页面的目录
│   ├── page-01.html
│   ├── page-02.html
│   └── ...
├── assets/             # 页面引用的资源文件（如 CSS、JS、图片）
│   ├── css/
│   ├── js/
│   └── images/
└── README.md           # 本文件
```

- **pages/**：所有独立 HTML 文件存放于此，每个文件对应一个独立页面。
- **assets/**：页面所需的静态资源，按类型分目录存放。
- **index.html**：可选的入口页面，用于导航或展示仓库内容。

## 页面归档说明

- 每个 HTML 页面均为自包含文件，尽量保持完整性和独立性。
- 页面可能包含外部链接或资源引用，但以本地资源优先。
- 归档内容不会主动更新，仅当需要修复或补充时进行修改。
- 不保证所有页面在所有浏览器或环境下完全兼容，但尽量遵循通用 Web 标准。

## 维护说明

- 本仓库由维护者不定期更新，新增或修改页面会记录在提交历史中。
- 如发现页面显示异常或链接失效，欢迎通过 Issues 反馈。
- 不接受未经审核的 Pull Request 直接修改内容，建议先提交 Issue 讨论。
- 仓库内容仅作为资料存档使用，不提供任何形式的担保或承诺。

## 使用方式

1. 克隆或下载本仓库到本地。
2. 直接使用浏览器打开 `pages/` 目录下的任意 HTML 文件即可查看。
3. 也可通过 GitHub Pages 或其他静态托管服务部署后在线访问。
